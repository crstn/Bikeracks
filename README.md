# Bikeracks
 
Data: [https://portal.opendata.dk/dataset/cykelstativer](https://portal.opendata.dk/dataset/cykelstativer)

Conda steps:

```
conda create --name bikeracks python=3.6
conda activate bikeracks
conda install psycopg2 flask
```
